My Story

Learning to Build the Web, One Property at a Time

I'm Fabian Munga — a developer in the making, currently deep in a structured frontend curriculum. Week 2 has been all about CSS — and I've fallen in love with how much power and creativity this language gives you.

From understanding the box model to building full responsive layouts, every day brings new "aha!" moments. Completing all 28 levels of Grid Garden and all 24 levels of Flexbox Froggy was a milestone I'm genuinely proud of.

My goal is to build professional, accessible, and beautiful web experiences — and this portfolio is step one.

  • CSS Grid & Flexbox — fully mastered
  • Mobile-first responsive design
  • Typography systems with CSS variables
  • Clean, semantic HTML5 structure
  • Accessibility-conscious development
